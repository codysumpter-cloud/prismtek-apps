# Team Audit Template

## Current format snapshot
- Date checked
- Sources used
- Active regulation or current public rules wording
- Supported formats
- Legality notes
- Meta notes
- Confidence
- Unresolved uncertainties

## Per-team audit
1. Original concept summary
2. Legality audit
3. Structural diagnosis
   - role compression
   - speed control
   - defensive backbone
   - pivoting / positioning
   - win conditions
   - matchup weaknesses
   - redundancy / anti-synergy
4. Keep / change decisions
5. Refined team
6. Why it is better now
7. Pilot notes
8. Threat checklist

## End block

- How to rerun this next month
