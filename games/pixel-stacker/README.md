# Pixel Stacker

Pixel Stacker is migrated from the Prismtek arcade inventory as a first-class browser game under `games/pixel-stacker/`.

## Source provenance

- Source inventory: `codysumpter-cloud/prismtek-site/src/arcade/games/PixelStackerGame.tsx`
- Migration tracker: `docs/games/prismtek-arcade-cross-platform-migration.md`

## Run

```bash
cd games/pixel-stacker
npm test
npm run dev
```

Open `http://localhost:4173`.

## Controls

- Space / click / tap: drop block
- Restart button: reset run
