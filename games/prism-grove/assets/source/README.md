# Prism Grove source assets

This folder is reserved for source-only character and garden assets used to build the Prism Grove runtime set.

Uploaded source packs for intake:

- `64 Pixel Female Characters 1.2 Body Update.png`
- `64 Pixel Female Characters.kra`
- `64 Pixel Female Characters 1.1 Hair Update.kra`
- `64 Pixel Male Character 1.0.kra`
- `Whimsypngs.zip`
- `SMS Oriental Garden Files.zip`
- `FreeEnvironment.zip`
- `Farm RPG FREE 16x16 - Tiny Asset Pack.rar`
- `FreeDemo.png.zip`
- `Back-Garde-Free.zip`

Krita files and third-party archives are source assets, not runtime dependencies. Runtime builds should use exported PNG layers, JSON manifests, and a license ledger before public release.

The current MVP uses procedural placeholder rendering so web, macOS, and iOS can run immediately while the licensed asset slicing pass happens safely.
