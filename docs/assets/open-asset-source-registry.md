## Required asset receipt

Before committing files from any source, create a receipt with:

- source URL
- source name
- asset title or pack name
- author/uploader
- exact license
- license URL or included license file
- attribution text if required
- file list
- destination path
- commercial-use status
- redistribution status
- modification status
- share-alike/copyleft status

No asset is approved just because the source appears in this registry. The registry is an intake map, not a blanket license grant.
