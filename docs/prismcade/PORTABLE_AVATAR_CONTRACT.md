# Prismcade Portable Avatar Contract

Status: foundation contract.

Prismcade avatars are reusable player identity kits for many small games.

Supported view families:

- side
- top_down
- low_top_down
- isometric

Required source kit pieces:

- base body
- hair
- face
- top clothing
- bottom clothing
- shoes
- accessory
- palette slots
- animation map
- view map
- provenance receipt

Support levels:

- none
- profile
- portrait
- palette
- mini
- full_view
- multi_view
- complete

Each game manifest should declare the avatar views it supports and the fallback it uses when a full avatar view is not ready.
