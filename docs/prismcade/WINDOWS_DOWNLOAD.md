# Prismcade Windows Download

Goal: provide one player-facing ZIP named Prismcade-Windows.zip.

Build command:

npm.cmd run prismcade:package:windows

Expected output:

dist/prismcade-windows/Prismcade-Windows.zip

Player flow:

1. Download Prismcade-Windows.zip.
2. Extract it.
3. Double-click Prismcade.exe to open the catalog.
4. Or double-click Pixel Fruit Arena.cmd to open Pixel Fruit Arena directly.
5. More direct game launchers are in the Launch Games folder.
