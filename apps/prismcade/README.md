# Prismcade Catalog

Static catalog/launcher prototype for the Prismcade platform loop.

## Run

From the repo root:

```bash
python -m http.server 4173
```

Open:

```txt
http://localhost:4173/apps/prismcade/
```

The catalog reads:

```txt
data/prismcade/game-manifests.json
```

## Validate

```bash
npm run prismcade:validate
```
