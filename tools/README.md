# Prismtek Tools

This directory contains repo-owned runnable tools for Prismtek workflows.

`prismtek-apps` is the canonical source for product tools. Public website repos may host, mirror, or link to these tools, but new tool implementation work should start here.

## Current tools

| Tool | Path | Purpose | Public host relationship |
| --- | --- | --- | --- |
| Sprite Sheet to GIF | [`tools/spritesheet-to-gif/`](spritesheet-to-gif/) | Convert pixel-art sprite sheets into animated GIFs locally in the browser. | `prismtek-site` may serve the deployed `/tools/spritesheet-to-gif/` route for `prismtek.dev`. |
