# Prismtek Apps Skills

This file is the human-facing entry point for repo-local operational skills.

## Source of truth

- `skills/README.md` documents the current skill set and expected skill format.
- `skills/index.json` is the machine-readable trigger/action registry.
- `skills/<name>/README.md` contains each skill's operator playbook.

Use this layer to keep Buddy, app-runtime, and fork-upgrade workflows discoverable across Prismtek Apps.
