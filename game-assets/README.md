# Game Assets (Reusable Library)

A curated, categorized library of reusable 2D/3D game art and audio assets for Prismtek games (e.g. `games/pixel-fruit-arena`, `games/tamernet-battle-sandbox`). Assets are grouped by type so any project in this monorepo can pull from a single shared source.

**252 asset files / packs · 146.35 MB total · 7 categories**

## Structure

Most entries are kept as their original archive (`.zip` / `.rar` / `.7z`) inside a category folder — this keeps the repository lean and browsable while preserving each pack exactly as published. Loose individual sprites are grouped into named subfolders. Unpack the pack you need into your project's local assets directory.

```
game-assets/
  characters/
  tilesets-environment/
  vfx/
  audio/
  ui/
  props-items/
  models-3d/
  _licenses/        # license documents shipped with individual packs
  README.md         # this file
```
