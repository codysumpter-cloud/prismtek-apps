# Game Assets (Reusable Library)

A curated, categorized library of reusable 2D/3D game art and audio assets for Prismtek games (e.g. `games/pixel-fruit-arena`, `games/tamernet-battle-sandbox`). Assets are grouped by type so any project in this monorepo can pull from one shared source.

**277 asset files / packs · 149.20 MB · 8 categories**

## How it's organized

Each pack is kept as its original archive (`.zip` / `.rar` / `.7z`) inside a category folder, which keeps the repo lean and browsable while preserving each pack exactly as published. Loose individual sprites are stored directly. Unpack the pack you need into your project's local assets directory. License documents that shipped loose are in `_licenses/`.

```
game-assets/
  characters/
  tilesets-environment/
  vfx/
  audio/
  ui/
  props-items/
  models-3d/
  misc/
  _licenses/
  README.md
```
