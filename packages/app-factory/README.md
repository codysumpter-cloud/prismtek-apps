# @prismtek/app-factory

Product scaffolding and generation primitives for `prismtek-apps`.

## Current role

This package contains the current app/template generation logic.

## Direction

This package should only remain here if app-factory behavior is truly part of the shipped product direction.

If it becomes legacy scaffolding or an incubator concept, it should be narrowed, moved, or retired instead of silently defining the repo identity.
