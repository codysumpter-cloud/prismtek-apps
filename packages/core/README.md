# @prismtek/core

Shared product-domain types and core contracts for `prismtek-apps`.

## Current role

This package holds shared types used across the current app/API/package layout.

## Direction

Keep this package focused on reusable product-domain contracts.

Examples:
- workspace types
- job and session types
- product-facing shared models

Avoid turning this package into a dump for unrelated utilities or cross-repo policy concerns.
