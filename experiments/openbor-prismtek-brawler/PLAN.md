# Evaluation Plan

## Purpose

Plan a tiny original Prismtek arcade prototype for evaluating this external engine reference.

## Player

- Name: Prism Runner
- Role: balanced starter
- Movement: walk, jump, dash
- Actions: basic combo and one special action
- Progression: rank badge and cosmetic receipt

## Test characters

- Scout: fast, low health
- Heavy: slower, higher health
- Final test character: durable closing encounter

## Stage

- Name: Neon Alley Test
- Length: one short lane
- Goals: movement, collision, pacing, scoring, and rank receipt
- Art rule: original Prismtek assets or approved provenance-recorded third-party assets only

## Decision criteria

Adopt the engine only if it improves iteration speed and can produce practical Android or Linux handheld test results.
