# OpenBOR Prismtek Evaluation

This folder defines a small Prismtek-owned evaluation for OpenBOR.

Use local reference import only:

```bash
node tools/reference-games/import-reference-game.mjs openbor
```

The checkout remains under `.external/reference-games/openbor/` and is not committed.

## Goal

Decide whether OpenBOR should be used for a future original Prismtek arcade game.

## Required notes

- engine import receipt
- original character plan
- original stage plan
- asset provenance record
- Android test notes
- Linux test notes
- final adopt/reference/reject decision
