# RGDS Test Plan

## Goal

Check whether an OpenBOR-based Prismtek experiment is practical on RGDS-style handheld targets.

## Android mode

Status: planned, not verified.

Checklist:

- Build or obtain an Android runtime artifact.
- Install on RGDS Android mode.
- Verify launch.
- Verify controller mapping.
- Verify frame pacing.
- Verify audio output.
- Record device, OS version, build SHA, and result.

## Linux mode

Status: planned, not verified.

Checklist:

- Build or obtain a Linux runtime artifact.
- Launch from RGDS Linux mode or compatible handheld Linux environment.
- Verify controller mapping.
- Verify display scaling.
- Verify frame pacing.
- Verify audio output.
- Record device, OS version, build SHA, and result.
