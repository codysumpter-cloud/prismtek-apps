# OpenBOR Prismtek Module Plan

## Working title

Prismtek Street Clash

## Purpose

Create a tiny original arcade-game module plan that can test OpenBOR without importing unreviewed external modules or assets.

## Player character

- Name: Prism Runner
- Role: balanced starter
- Movement: walk, jump, dash, basic combo
- Special: short-range energy burst
- Progression: rank badge and cosmetic unlock receipt

## Enemy set

- Scout: fast, low health, teaches spacing
- Bruiser: slow, high health, teaches recovery timing
- Boss test: one durable end encounter with a readable wind-up

## Test stage

- Name: Neon Alley Test
- Length: one short lane
- Goals: movement, collision, spawn pacing, score/rank receipt
- Art rule: original Prismtek assets or approved provenance-recorded third-party assets only

## Decision criteria

Adopt OpenBOR only if it gives faster content iteration than the existing browser arcade stack and can produce a practical Android or Linux handheld test build.
